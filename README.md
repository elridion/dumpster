# Dumpster

Simple and easy to use binary-dumps.

## Installation

The package can be installed by adding `dumpster` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:dumpster, "~> 0.0.1"}
  ]
end
```
